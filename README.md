### Setup
* Clone this repository
* Run make to build the main application.
* Run make test to build and execute all unit tests.
* Run make clean to remove all build artifacts.
* Run make docs to generate/update diagrams in the diagrams folder (requires PlantUML).

### Test Framework Setup
This project uses the Unity test framework (MIT License) for unit testing.

For more details on test structure and usage, see the docs folder.
